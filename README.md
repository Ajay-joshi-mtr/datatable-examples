# datatable-examples
Using DataTables plug-in for jQuery https://datatables.net

Having functionalities such as : 
1. Search
2. Sort
3. Expand Row
4. Edit Row (Inline)
5. Edit Multiple Rows (Inline)
6. Fix Column/s
7. Custom Coloring
8. Column Visibility
9. Multiple Header Rows (With/Without Colspan)

Demo : http://eunoia.gq
